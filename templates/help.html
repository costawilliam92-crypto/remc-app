Help & Documentation

Getting Started

Welcome to the REMC Web Application! This web-based version allows you to access your project management system from any device with a web browser, including iOS devices.

Key Features:
  • Cross-Platform: Works on iOS, Android, Windows, Mac, and any device with a web browser
  • Responsive Design: Optimized for mobile devices and tablets
  • Real-time Data: Access to all your projects and email tracking data
  • Touch-Friendly: Designed for touchscreen interaction
Navigation Guide
Dashboard

Overview of all project statistics, quick search, and recent projects. Your main starting point.

Projects

Browse all projects with search and filter capabilities. Switch between list and grid views.

Email Tracking

View all tracked emails with advanced filtering by project, date, and content.

Project Details

Detailed view of individual projects including all related emails and attachments.

Mobile & iOS Tips
For iOS Devices:
  • Add to Home Screen: In Safari, tap the share button and select "Add to Home Screen" for app-like experience
  • Full Screen: The web app will run in full-screen mode when added to home screen
  • Touch Navigation: Swipe and tap gestures work throughout the interface
  • Search: Use the search boxes to quickly find projects and emails
Performance Tips:
  • Close other browser tabs for better performance
  • Refresh the page if data seems outdated
  • Use Wi-Fi for faster loading of large project lists
  • The app works offline for recently viewed content
Features Guide

Search across projects and emails using various criteria:

  • Project names, IDs, and descriptions
  • Client names and locations
  • Email subjects and content
  • Sender names and addresses

Search results update in real-time as you type.

Email tracking features include:

  • View all emails associated with projects
  • Filter by project, date, or content
  • View email attachments and details
  • Track both received and sent emails

Note: Email tracking requires the desktop Outlook integration to be set up.

Project management capabilities:

  • View residential and commercial projects
  • Browse project details and statistics
  • Switch between list and grid view modes
  • Filter projects by type and status
  • Access project-specific email communications
